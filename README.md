# Football Market Bot ⚽📊

Real-time probabilistic football market scanner using **iSportsAPI** with Telegram notifications.

## Features

- 🔄 **Live Data** - Fetches live scores & in-play stats from iSportsAPI
- 🧮 **Advanced Probability Engine** - Poisson + Dixon-Coles models with real-time xG projection
- 📱 **Telegram Notifications** - Top 5 high-probability markets delivered instantly
- 📝 **Markdown Reports** - Detailed analysis saved locally
- ⚡ **Async Architecture** - Non-blocking HTTP with proper rate limiting
- 🛡️ **Type Safe** - Full Pydantic validation & MyPy strict mode
- ✅ **Tested** - 80%+ coverage with unit & integration tests
